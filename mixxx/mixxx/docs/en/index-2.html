2.3 Compilation and installation

In order to compile and install Mixxx on your system, type the following in the base directory of the Mixxx distribution: 

% ./configure
% make
% make install

Since Mixxx uses 

autoconf
you should have not trouble compiling it. Should you run into problems please report them to the the author at Tue and Ken Haste Andersen
